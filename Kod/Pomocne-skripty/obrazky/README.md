# Obrázky
Tato složka obsahuje skripty pro **konvertování PNG obrázků na C skripty pro použití** v STM32CubeIDE. 

## [png-converter.py](./png-converter.py)
Konvertuje libovolný obrázek ve formátu `.png` na soubor `.c`, který lze použít v STM32CubeIDE. 

### Použití
```bash
python3 png-converter.py input.png image_name output.c
```
